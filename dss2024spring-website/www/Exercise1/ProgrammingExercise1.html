2. Sequential data denoise using structured learning

2.1 The Problem

First, we generate the data for a simple sequential data denoising problem. 

x
import numpy as np
import matplotlib.pyplot as plt
from sklearn.model_selection import train_test_split


n_samples = 500


d = np.array([12, 12, 11, 11, 10,  9,  8,  8,  7,  6,  6,  6,  7,  8,  8,  8,  6, 5,  4,  3,  3,  3,  2,  1,  0,  1,  3,  4,  5,  6,  8,  8,  9,  9, 10, 11, 12, 13, 14, 14, 14, 15, 15, 15, 15])


n_nodes = d.shape[0]
n_states = np.unique(d).shape[0]
n_features = n_states + 1 # add bias


y = np.repeat(d[np.newaxis,:], n_samples, axis=0)


# add noise
data = y + (np.random.rand(n_samples, n_nodes)-0.5)*5

The generated data can be visualized as follows:

The input are N=500 noisy data sequences of length l=45 such as the daily temperature measurements from multiple sensors. Our goal is to recover the denoised sequence y=[y1,…,yl],yi∈Y for each noisy observation sequence x=[x1,…,xl], xi∈R . Assume that each yi is a discrete variable with 16 states (Y={0,…,15}).

2.2 Structured Prediction

To predict y given x using structured prediction, we need to solve

y∗=arg⁡maxy∈Y⁡f(x,y),

where f is a compatibility function (a.k.a. potential function) that says how well y fits the input x.

Pystruct takes f to be a linear function of some parameters w and a joint feature function of x and y:

f(x,y)=wTjoint_feature(x,y).

Here vector w are parameters learned from data, and joint_feature() is defined by the user-specified structure of the model. In this exercise, we adopt a linear chain CRF model (as shown above) build the correspondence between input x and output y.

we can model this problem using a linear chain CRF with l nodes: (e.g. l=4 in the following figure)

 

In linear chain CRF, the compatibility function can be written as the sum of a unary term and a pairwise term:

f(x,y)=⟨wu,∑i=1lΦ(xi)⊗Λc(yi)⟩+⟨wp,∑i=1l−1Λc(yi)⊗Λc(yi+1)⟩

where

  • Φ(xi)∈Rd is the feature vector of xi
  • Λc(yi)∈{0,1}|Y| is the one-hot vector of label yi
  • u⊗v=uvT is the outer product operator
  • wu∈Rd×|Y| and wp∈R|Y|×|Y| are the unary and pairwise parameters to be learned.

The unary term summarizes the interaction between features and labels; The pairwise term summarizes the interaction between labels of adjacent nodes. We can write f(x,y) as a linear function of w=[wuwp] as

f(x,y)=⟨w,[∑i=1lΦ(xi)⊗Λc(yi)∑i=1l−1Λc(yi)⊗Λc(yi+1)]⟩

 

In the sequence denoising example, we define input feature Φ(xi) as a vector of dimension |Y|+1:

Φ(xi)=−[|xi−1|0.1|xi−2|0.1⋮|xi−|Y||0.10.1]

The last element is a constant bias term.

Use the following code to generate the feature matrices for all input sequences and visualize the features: 

X = np.array( [ [ [ -abs(i-j)**0.1 for j in range(n_states)] for i in dd ] for dd in data] )


# add constant features for bias
X = np.array( [np.hstack((X[i], 0.1*np.ones((X[i].shape[0],1)))) for i in
range(X.shape[0])] )


# visualize features of the first input 
plt.matshow(X[0].transpose())
plt.title('feature function f(x,y) of sample 0')
plt.ylabel('states (y)')
plt.xlabel('nodes (x)')
plt.show()

Next we prepare the training and testing data and initialize the ChainCRF model. Note that you only need to provide the input features and labels. For details on the joint_feature implementation, see the source code for ChainCRF and GraphCRF. 



from sklearn.model_selection import train_test_split


from pystruct.inference import inference_dispatch
from pystruct.models import ChainCRF
from pystruct.learners import SubgradientSSVM


X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.4, random_state=0)


model = ChainCRF()

Fit the model with the training data using Subgradient SSVM method:



ssvm = SubgradientSSVM(model, C=5.0,  max_iter=100)
ssvm.fit(X_train, y_train)

Let's extract the fitted parameters wu and wp from the model. In the SSVM model, all parameters are stored in an 1D array w . First make sure its length equals to d×|Y|+|Y|2=17×16+162=528

# visualize parameter size 
print(ssvm.w.shape)

We can visualize the unary parameters:

plt.matshow(ssvm.w[:n_features * n_states].reshape(n_states,n_features))
plt.title("Unary parameters of the chain CRF.")
plt.yticks(np.arange(n_states))
plt.xticks(np.arange(n_features))
plt.ylabel('states') 
plt.xlabel('features')
plt.colorbar()
plt.show()

Visualize the learned pairwise parameters:

plt.matshow(ssvm.w[n_features * n_states:].reshape(n_states, n_states))
plt.title("Pairwise parameters of the chain CRF.")
plt.xticks(np.arange(n_states))
plt.yticks(np.arange(n_states))
plt.colorbar()
plt.show()

Finally, we compute the prediction results on the testing set and visualize the output.

print ("Test score:",ssvm.score(X_test, y_test))
X_test_predict = np.array(ssvm.predict(X_test))


# visualize the first four prediction results
fig, axes = plt.subplots(2,2, figsize=(12,6))
shf = np.arange(X_test.shape[0])
